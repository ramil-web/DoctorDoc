# DoctorDoc Backend

DoctorDoc - это REST API сервис на языке Go для управления медицинскими данными. Проект построен с использованием микросервисного подхода, поддерживает JWT-аутентификацию и работу с базой данных PostgreSQL.

## Основные возможности

* Безопасная авторизация: Реализована через JWT (JSON Web Tokens) с использованием кастомных Middleware.
* Защита от атак (Rate Limiting): Встроенный ограничитель запросов на базе Redis для предотвращения перегрузок.
* Работа с данными: Интеграция с PostgreSQL для хранения профилей врачей и медицинских записей.
* Файловое хранилище: Обработка и загрузка документов в директорию /uploads.
* Чистая архитектура: Разделение на слои (cmd — запуск, internal — логика, migrations — БД).
* Docker: Полная контейнеризация для быстрого развертывания.

## Технологический стек

* Язык: Go (Golang)
* БД: PostgreSQL
* Кэширование: Redis
* Контейнеры: Docker и Docker Compose
* API: RESTful API

## Структура проекта

cmd/             # Точки входа (main.go)
internal/        # Внутренняя логика (бизнес-правила, репозитории)
migrations/      # SQL скрипты для миграции базы данных
uploads/         # Папка для загруженных документов
docs/            # Документация проекта
.env.example     # Пример файла конфигурации
docker-compose.yml

## Быстрый старт


Настройка окружения:
Создайте файл .env и заполните свои данные (БД, порты, секреты):
cp .env.example .env

Запуск через Docker:
docker-compose up --build -d

## Примеры API эндпоинтов

Метод | Путь | Описание
--- | --- | ---
POST | /api/v1/auth/register | Регистрация нового пользователя
POST | /api/v1/auth/login | Вход и получение токена
GET | /api/v1/user/profile | Получение данных профиля (нужен JWT)
POST | /api/v1/upload | Загрузка медицинского файла
